# @coreo/lib

Librairie partagée de la couche métier.

## Contenu

- `env.ts` : validation runtime des variables d’environnement avec Zod
- `config.ts` : configuration exposée au frontend
- `directus.ts` : client Directus typé
- `seo.ts` : helpers SEO et metadata
- `modules.ts` : activation des modules métier via env
