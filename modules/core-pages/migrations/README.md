# Migrations core-pages

Ce dossier contient des migrations SQL ou des notes pour synchroniser la structure Directus avec PostgreSQL.

## Usage

- Ajoutez ici les scripts SQL si vous souhaitez versionner des modifications de schéma explicites.
- Directus gère les collections, mais il est utile de conserver des fichiers `migrations/*.sql` pour les contraintes ou index personnalisés.
